# 朵朵 10 岁生日邀请函

这是一个为朵朵 10 岁生日定制的 H5 邀请函应用。它具备精美的“魔法森林”主题动画、背景音乐和一键导航功能，旨在通过微信分享给朋友和家人，带来更具互动性和仪式感的体验。

## ✨ 特性

- **魔法森林主题**: 柔和的马卡龙色系，带有闪烁和漂浮的魔法元素。
- **互动开场动画**: 点击“打开邀请函”按钮，平滑过渡到内容页。
- **关键信息展示**: 清晰显示生日人物、日期、地点。
- **成长照片墙**: 展示朵朵成长瞬间的图片画廊。
- **背景音乐**: 可切换播放/暂停的温馨背景音乐。
- **一键导航**: 点击按钮即可跳转至地图应用进行导航。
- **移动端优化**: 针对微信内置浏览器和各类移动设备进行响应式优化。

## 🚀 运行项目

### 前提条件

确保您的系统已安装 Node.js (推荐 v18+) 和 npm。

### 1. 安装依赖

在项目根目录 `/home/website/birthday` 下运行：

```bash
npm install
```

### 2. 开发模式运行

项目将在开发服务器上运行，通常在 `http://localhost:5173/` 可访问。

```bash
npm run dev
```

### 3. 构建生产版本

此命令会将项目打包成静态文件，生成在 `dist` 文件夹中。这些文件可以直接部署到任何静态文件托管服务。

```bash
npm run build
```

## 🛠 技术栈

- **前端框架**: React (TypeScript)
- **构建工具**: Vite
- **样式**: Vanilla CSS (CSS Modules)
- **动画库**: Framer Motion
- **图标库**: Lucide-React

## 💖 定制

- **修改图片**: 替换 `src/components/InvitationPage.tsx` 中的图片 `src` 属性，或在 `public` 目录下放置图片后引用。
- **更新音乐**: 替换 `src/components/InvitationPage.tsx` 中 `<audio>` 标签的 `src` 属性为您的音乐 URL。
- **调整文字**: 直接修改 `src/components/InvitationPage.tsx` 中的文本内容。
